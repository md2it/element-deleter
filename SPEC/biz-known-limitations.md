# ИЗВЕСТНЫЕ ОГРАНИЧЕНИЯ

- **Выделение iframe отличается** от выделения иных элементов:
   - iFrame захватывается целиком
      - Т.к. ограничение платформы
      - Внедрение в сам iFrame считаю нежелательным
   - Есть визуальное отличие выделения
      - Т.к. разные обработчики
      - Не влияет на работоспособность
      - Унификация выделения не несёт функциональной пользы
- **Восстановление SVG path** иногда некорректно:
   - Действительно баг, влияющий на функциональность
   - Попытки исправить заняли много времени
   - Значимость бага низкая, т.к. сценарий редкий
