# Restoring an element

- By a button in the notification:
   - Restores the corresponding element
- By keyboard shortcut:
   - Restores the last deleted element
   - Repeated use of the shortcut restores elements in reverse deletion order
- Triggers a restoration notification
- Conserve client memory:
   - "Remembering" works only within the current page
   - Closing or reloading the page clears the memory
