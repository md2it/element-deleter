ELEMENT DELETER
移除页面干扰元素，用于测试、截图、设计检查或清爽阅读。

Element Deleter 可以快速清理网页上碍事的内容：横幅、弹窗、固定页眉、小组件、多余区块、iframe 和其他分散注意力的元素。

它适合前端开发、测试和设计人员：在没有视觉干扰的情况下检查页面，制作干净截图，查看布局效果，或移除遮挡内容的元素。日常使用时，也能让页面更容易阅读、查看和保存。

悬停、点击，元素就会消失。删错了？可以恢复。

主要功能
- 只需几次点击即可删除页面元素
- 恢复已删除的元素
- 在删除模式开启时可多次撤销删除
- 通过右键菜单删除元素
- 支持 iframe 和嵌入内容
- 删除后显示清晰通知
- 轻量、简单
- 仅使用本地设置

隐私
- 不收集数据
- 不跟踪用户
- 不发送网络请求
- 更改仅在当前页面本地生效
- 重新加载页面会恢复原始内容
