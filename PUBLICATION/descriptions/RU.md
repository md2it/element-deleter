ELEMENT DELETER
Убирайте лишние элементы со страницы: для проверки, скриншотов и спокойного чтения.

Element Deleter быстро очищает страницу от всего, что мешает: баннеров, попапов, лишних шапок, виджетов, лишних блоков, iframe и других элементов.

Полезно фронтендерам, тестировщикам и дизайнерам: быстро проверить страницу без мешающих блоков, подготовить чистый скриншот, посмотреть вариант макета или убрать перекрывающий элемент. Обычным пользователям — читать, смотреть и сохранять страницы без визуального шума.

Навели, кликнули — элемент исчез. Ошиблись — восстановили.

ВОЗМОЖНОСТИ
- Удаление элементов страницы в несколько кликов
- Восстановление удалённых элементов
- Многократная отмена удалений, пока активен режим удаления
- Удаление элементов через контекстное меню
- Работа с iframe и встроенным содержимым
- Понятное уведомление после удаления
- Лёгкое и простое расширение
- Только локальные настройки

ПРИВАТНОСТЬ
- Данные не собираются
- Нет трекинга
- Нет сетевых запросов
- Изменения применяются только локально на текущей странице
- После перезагрузки страницы исходное содержимое восстанавливается
